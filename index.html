Introduction

HTML stands for HyperText Markup Language. It is what is used to provide a framework for some of your favorite sites. Nothing you see on the web would be there without HTML. This documentation will give you a guide to it. Don't forget to always place !DOCTYPE html and the beginning of every web page like the following.

!DOCTYPE html

My journey of learning HTML has been an arduous one, but it has been well worth it. I wish you the best on your journey as well.

HTML and CSS

We have already covered what HTML is from a high-level. Another thing that goes hand in hand with HTML is CSS. If you think of HTML as the framework, CSS gives the page some style. Our web pages would be rather boring without some type of style behind them. That is where CSS comes in. An example of CSS is below.

body { text-align: center; } p { background-color: #eee; }

There are different ways to use CSS. You can include it directly into your HTML code, or you can create a separate place for it or a, "stylesheet." This is by far the most common method. It is very easy to neglect CSS, but don't ever forget that a disrespected CSS sheet will create a disrepected website.
